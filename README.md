# Strategic Minds AI Business Builder

Enterprise-grade governed AI business builder control plane.

This repo supports the Strategic Minds AI workflow:

PLAN -> DISCOVERY -> BRAND -> APPROVAL -> DOCS -> BUILD -> VALIDATE -> RELEASE

## Purpose

Create a repeatable system for AI-enhancing existing businesses, rebranding businesses, creating new businesses, and automating business workflows.

## Governance

- AUTO BUILDER MCP is the governance and validation brain.
- Base44 is the implementation/super-agent layer inside approved boundaries.
- Vercel Workflow uses a 5-minute cron trigger.
- Supabase is the backend and memory layer.
- Google Drive is the human-readable source-truth and delivery layer.
- GitHub is the code, schema, packet, and canonical technical docs layer.

## Protected Actions

Production deploys, secrets, database migrations, customer communications, live social publishing, payments, destructive actions, and spend require explicit approval.
