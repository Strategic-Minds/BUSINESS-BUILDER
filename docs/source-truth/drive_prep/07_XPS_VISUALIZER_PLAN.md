# XPS Visualizer and Product Data Plan

## Goal
Implement a floor visualizer experience for epoxy/decorative concrete customers without copying protected code or assets.

## Phase 1
Use public Xtreme Polishing Systems visualizer/link as the integration reference:
https://xtremepolishingsystems.com/pages/flake-visualizer

## Phase 2
Create a lawful custom visualizer using authorized assets:
- room scenes
- flake blends
- metallic systems
- polished concrete finishes
- product recommendations
- material estimate hooks
- proposal attachment

## Required UX
- Select room type.
- Select finish type.
- Select color/flake/metallic.
- Preview finish.
- Save selection to proposal.
- Add material recommendation.
- Push to client dashboard.

## Required Approval
Use of XPS logos, product images, color charts, or protected assets beyond public linking requires authorization or confirmed license.
