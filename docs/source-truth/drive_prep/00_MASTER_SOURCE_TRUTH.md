# AI Business Factory - Master Source Truth

## Mission
Build a universal AI Business Factory that can create vertical business-in-a-box systems for service industries, starting with epoxy, decorative concrete, and polished concrete.

## Primary Outcome
When the operator approves, the system should generate a preview-ready, tested, scored, documented, and approval-gated business system overnight.

## First Vertical
Epoxy / decorative concrete / polished concrete contractors and distributors.

## Core Promise
Turn every lead into a qualified opportunity, proposal, payment gate, scheduled job, documented project, Google review, social proof asset, and SEO signal with minimal office staff.

## Required Systems
- Website and package selector
- Intake form and dashboard assignment
- Proposal and visual MVP generation
- Stripe/Square payment gates
- WhatChimp WhatsApp workflows
- Google Workspace docs, sheets, forms, calendar, tasks
- GitHub branch and PR workflow
- Vercel preview deployment workflow
- Vercel 5-minute cron
- QA/headless validation
- SEO persistence engine
- XPS visualizer integration plan
- Release gate and rollback plan
