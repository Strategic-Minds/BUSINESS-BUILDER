Create a compounding local visibility system that turns every client build and every completed job into an SEO asset.

- Local service page generator.
- City/location page generator.
- Schema generator.
- Metadata optimizer.
- Internal linking engine.
- Image alt text engine.
- Before/after case study generator.

- Review request automation.
- Search Console watchlist.
- Competitor page tracker.
- Content calendar.

## Guardrail
Do not promise guaranteed rankings. Position as a compounding visibility engine.
